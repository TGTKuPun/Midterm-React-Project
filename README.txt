========= PROJECT SETUP GUIDE ============

1. DOWNLOAD THE PROJECT

### Option 1: Using Git
If you have Git installed, you can clone the repository using the command below:
"git clone https://github.com/TGTKuPun/Midterm-React-Project.git"


### Option 2: Download ZIP via TDTU e-learning or GitHub
### For Github
- Visit: https://github.com/TGTKuPun/Midterm-React-Project.git
- Click the "Code" button, then select "Download ZIP"
- Extract the ZIP file to any folder on your machine

2. RUN THE PROJECT

### Option 1: Using Node.js

**Requirements:**
- Node.js and npm installed

**Steps:**
1. Open your terminal or command prompt
2. Navigate to the project directory
3. Install dependencies: npm install
4. Start the development server: npm run dev
5. Open your browser and go to: http://localhost:5173 


### Option 2: Using Docker

**Requirements:**
- Docker and Docker Compose installed

**Steps:**
1. Open your terminal or command prompt
2. Navigate to the project directory
3. Run the following command: docker-compose up
4. Open your browser and go to: http://localhost:5173
